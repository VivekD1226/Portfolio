# 🚀 Vivek Dalimbkar — Portfolio

A modern, minimal, and professional personal portfolio website built with pure HTML, CSS, and JavaScript.

🔗 **Live:** [vivekd1226.github.io/Portfolio](https://vivekd1226.github.io/Portfolio/)

---

## ✨ Features

- **Dark / Light Mode** — Smooth toggle with localStorage persistence
- **Responsive Design** — Mobile-first layout with hamburger menu
- **Scroll Animations** — Elements fade in as they enter the viewport
- **Micro-Interactions** — Hover effects, button lifts, and border glows
- **Contact Form** — Visual submit feedback with loading and success states
- **Resume Download** — One-click PDF download
- **SEO Optimized** — Meta tags, Open Graph, semantic HTML

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| **Structure** | HTML5, Semantic Markup |
| **Styling** | Vanilla CSS, CSS Variables, Custom Properties |
| **Interactivity** | Vanilla JavaScript, Intersection Observer API |
| **Typography** | Inter, JetBrains Mono (Google Fonts) |
| **Hosting** | GitHub Pages |

## 🖥️ Sections

1. **Hero** — Bold intro with CTA buttons and social links
2. **About** — Professional summary with highlights
3. **Skills** — Categorized tech stack (Frontend, Backend, Database, Tools, AI & Cloud)
4. **Projects** — Featured project cards with tech tags and links
5. **Experience** — Timeline of certifications, hackathons, and achievements
6. **Contact** — Contact form with social links

## 🚀 Getting Started

Clone the repo and open `index.html` — no build step needed.

```bash
git clone https://github.com/VivekD1226/Portfolio.git
cd Portfolio
# Open index.html in your browser
```
